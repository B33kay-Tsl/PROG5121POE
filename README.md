# PROG5121POE
1. Open the project in NetBeans
2. Right-click Main.java
3. Click Run File
4. Type in your details when the program asks

WHAT YOU NEED TO ENTER:
- First name
- Last name
- Username  (must have an underscore and be max 5 characters)
- Password  (must have 8+ characters, a capital letter,
             a number, and a special character)
- Cell number (must include international code e.g. +27)

HOW TO RUN THE TESTS:
1. Right-click LoginTest.java
2. Click Test File
3. A green bar means all tests passed

FILES IN THE PROJECT:
- Login.java     : Handles registration and login logic
- Main.java      : Runs the program and takes user input
- LoginTest.java : Contains all JUnit tests
